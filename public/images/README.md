# 圖片放置說明 (Image Slots)

本網站所有圖片都採用「槽位 (Slot)」設計：
程式碼已經寫好對應路徑，你只要把圖片檔放進對應資料夾、用對的檔名，
重新整理頁面就會自動顯示。**還沒放圖時，畫面會以墨綠漸層佔位，不會破版。**

## 檔案格式建議
- 格式：`.jpg`（照片）或 `.webp`（更小更快，推薦）
- 若改用 `.webp`，記得同步把 `content.ts` 裡的副檔名改成 `.webp`
- 單張建議壓到 300–600KB 以內，載入較快

## 新增更多案例
1. 在 `public/images/cases/` 下新建一個資料夾，例如 `taipei-xxx/`
2. 在 `src/lib/content.ts` 的 `caseStudies` 陣列複製一筆、改 `slug` 與內容
3. 若要做完整詳情頁，參考 `haijingCase` 物件結構新增資料，
   並在 `src/app/cases/[slug]/page.tsx` 的 `CASES` 對照表登錄
